# Payment Checkout App

A 5-step payment checkout application built with a modern stack, following Hexagonal Architecture and Railway Oriented Programming (ROP).

## Tech Stack
-   **Frontend**: React 19.1, Vite, Redux Toolkit, Tailwind CSS
-   **Backend**: NestJS, Prisma, PostgreSQL
-   **Integration**: Wompi Sandbox for payment tokenization and transactions

## Project Structure
-   `backend/`: NestJS application
-   `frontend/`: React application
-   `docs/`: Specification and implementation plan
-   `.agents/`: Development skills and automation tools
